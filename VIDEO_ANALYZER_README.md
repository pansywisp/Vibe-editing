# Qwen3-VL 视频分析工具

本工具使用 Qwen3-VL 视觉语言模型分析视频素材，直接将视频文件发送给API进行分析，包括视频的声音内容。

## 功能特点

- 支持视频和图片分析
- 直接分析整个视频（包括声音）
- 无需手动提取关键帧，API内部自动处理
- 支持大视频文件处理
- 详细的错误处理和结果输出

## 环境要求

- Python 3.7+
- Pillow
- requests

## 安装依赖

```bash
pip3 install pillow requests
```

## 使用方法

### 分析视频

```bash
python3 qwen3vl_video_analyzer.py /path/to/video.mp4 --api-key "your-api-key"
```

### 分析图片

```bash
python3 qwen3vl_video_analyzer.py /path/to/image.jpg --api-key "your-api-key"
```

### 自定义分析提示词

```bash
python3 qwen3vl_video_analyzer.py /path/to/video.mp4 --api-key "your-api-key" --prompt "分析这个视频的内容、风格、声音和情感表达"
```

## 技术原理

1. **视频处理**：
   - 直接读取视频文件
   - 编码为 base64 格式
   - 发送给 Qwen3-VL API 进行分析

2. **分析过程**：
   - API 内部自动处理视频内容
   - 同时分析视频的视觉和声音部分
   - 返回综合分析结果

3. **大视频处理**：
   - 利用 API 内部的视频处理能力
   - 无需本地提取关键帧
   - 减少本地计算资源占用

## 注意事项

1. **视频大小**：虽然 API 支持大视频分析，但过大的视频可能会导致请求超时或失败

2. **API调用限制**：视频分析会消耗更多的 API 配额，请注意使用限制和费用

3. **网络连接**：确保网络连接稳定，视频分析需要较长时间

4. **视频格式**：支持的视频格式包括：mp4、avi、mov、wmv、flv、mkv

## 故障排查

- **视频编码失败**：检查视频文件是否损坏，格式是否支持
- **API调用失败**：检查API密钥是否正确，网络连接是否稳定
- **请求超时**：对于大视频，可能需要增加超时时间
- **内存占用过高**：对于特别大的视频，可能需要分批处理
