# Qwen3-VL 视频分析工具

本工具使用 Qwen3-VL 视觉语言模型分析视频素材，通过提取关键帧的方式处理大视频文件。

## 功能特点

- 支持视频和图片分析
- 自动提取视频关键帧
- 可配置提取帧数和间隔
- 支持大视频文件处理
- 详细的错误处理和结果输出

## 环境要求

- Python 3.7+
- OpenCV (`cv2`)
- NumPy
- Pillow
- requests

## 安装依赖

```bash
pip3 install opencv-python numpy pillow requests
```

## 使用方法

### 分析视频

```bash
python3 qwen3vl_video_analyzer.py /path/to/video.mp4 --api-key "your-api-key"
```

### 分析图片

```bash
python3 qwen3vl_video_analyzer.py /path/to/image.jpg --api-key "your-api-key"
```

### 自定义参数

```bash
# 自定义提取帧数和间隔
python3 qwen3vl_video_analyzer.py /path/to/video.mp4 --api-key "your-api-key" --max-frames 5 --frame-interval 60

# 自定义分析提示词
python3 qwen3vl_video_analyzer.py /path/to/video.mp4 --api-key "your-api-key" --prompt "分析这个视频的内容和风格"
```

## 技术原理

1. **视频处理**：
   - 使用 OpenCV 打开视频文件
   - 按照指定间隔提取关键帧
   - 调整帧大小并编码为 base64 格式

2. **分析过程**：
   - 对每个关键帧单独分析
   - 汇总所有帧的分析结果
   - 生成综合视频分析报告

3. **大视频处理**：
   - 只提取有限数量的关键帧
   - 调整帧大小以减少API请求大小
   - 分批处理，避免内存占用过高

## 注意事项

1. **视频大小**：对于特别大的视频，可能需要调整 `--max-frames` 和 `--frame-interval` 参数以平衡分析质量和处理速度

2. **API调用限制**：每个关键帧都会触发一次API调用，请注意API使用限制和费用

3. **网络连接**：确保网络连接稳定，特别是分析多个关键帧时

4. **视频格式**：支持的视频格式包括：mp4、avi、mov、wmv、flv、mkv

## 故障排查

- **视频无法打开**：检查视频文件是否损坏，格式是否支持
- **关键帧提取失败**：尝试调整 `--frame-interval` 参数
- **API调用失败**：检查API密钥是否正确，网络连接是否稳定
- **内存占用过高**：减少 `--max-frames` 参数值
